# Dynamic-CRUD-Application-using-PHP-AJAX

A dynamic Student Management System built using PHP, MySQL, AJAX, and jQuery. This project demonstrates how to perform CRUD (Create, Read, Update, Delete) operations without page reload, providing a smooth and interactive user experience.

# Features
Add new student records
View all student records
Update existing student details
Delete records with confirmation
AJAX-based operations (No Page Refresh)
jQuery event handling
MySQL database integration
Responsive and user-friendly interface
Success and error message notifications
Modal popup for editing records

MySQL
AJAX
jQuery
